隐私政策

最后更新日期：2026年2月2日

1. 简介

欢迎使用 Manna（"我们"、"我们的"或"本应用"）。本隐私政策说明我们在您使用离线圣经小组件应用时如何处理您的信息。

Manna 以隐私为核心原则设计。我们致力于保护您的隐私，确保您的个人信息安全。

2. 我们收集的信息

我们不收集、存储或传输任何个人信息。

Manna 是一款完全离线的应用，具有以下特点：

  • 不需要用户注册或登录
  • 不收集任何个人数据
  • 不跟踪您的使用情况或行为
  • 不使用分析或跟踪工具
  • 不访问您的通讯录、位置或其他设备数据

3. 信息使用方式

由于我们不收集任何信息，因此我们不使用、共享或处理任何个人数据。

4. 本地数据存储

所有应用数据都存储在您的设备本地，包括：

  • 从 BibleGateway（和合本）和 GitHub Bible Database 下载的圣经内容
  • 您的主题偏好设置
  • 您的字体样式选择
  • 您的自定义背景图片

这些数据永远不会离开您的设备，我们或任何第三方都无法访问。

5. 图片分享功能

Manna 允许您创建和分享圣经经文图片。当您使用分享功能时：

  • 图片在您的设备本地生成
  • 图片不会上传到我们的服务器（我们没有任何服务器）
  • 分享完全由 iOS 的原生分享系统处理
  • 您可以控制图片的分享位置和方式

6. 应用内购买

Manna 提供一次性购买以解锁高级功能，包括：

  • 额外主题
  • 自定义字体样式
  • 自定义背景图片

所有购买交易均通过 Apple App Store 处理。我们无法访问您的支付信息。有关 Apple 如何处理您的支付数据的信息，请参阅 Apple 的隐私政策。

7. 第三方服务

Manna 不集成任何第三方服务、分析平台或广告网络。

7.1 圣经内容来源

Manna 中的圣经内容来源于以下开源资源：

  • BibleGateway - 中文和合本简体版（CUVS）
    来源：https://www.biblegateway.com/versions/Chinese-Union-Version-Simplified-CUVS/
    中文和合本属于公共领域，可自由使用。
  • GitHub - Bible Database (scrollmapper/bible_databases)
    来源：https://github.com/scrollmapper/bible_databases
    此开源数据库提供结构化的圣经文本数据，可用于个人和商业用途。

此内容在应用安装时下载一次并存储在您的设备本地。在正常应用使用期间，不会与这些服务保持持续连接。我们不会跟踪或监控您的圣经阅读活动。

8. 儿童隐私

Manna 的年龄分级为 4+，适合所有年龄段的用户使用。由于我们不收集任何个人信息，因此没有关于儿童数据的特殊考虑。

9. 数据安全

由于所有数据都存储在您的设备本地，我们不收集或传输任何信息，因此您的数据安全取决于您设备的安全措施。我们建议：

  • 使用强密码保护设备
  • 保持 iOS 系统更新
  • 使用设备加密（iOS 默认启用）

10. 国际数据传输

由于我们不收集或传输任何数据，因此不存在国际数据传输。

11. 您的权利

根据各种隐私法律（包括 GDPR、CCPA 等），您对个人数据拥有某些权利。但是，由于我们不收集任何个人信息，这些权利不适用于我们的应用。

您可以完全控制设备上存储的所有应用数据。您可以：

  • 随时删除应用以移除所有本地数据
  • 在应用内重置应用设置
  • 通过您的 Apple ID 管理购买历史记录

12. 隐私政策变更

我们可能会不时更新本隐私政策。我们将通过以下方式通知您任何变更：

  • 在本页面发布新的隐私政策
  • 更新"最后更新日期"
  • 对于重大变更，提供应用内通知

13. 法律合规

本隐私政策符合以下法规：

  • 通用数据保护条例（GDPR）- 欧盟
  • 加州消费者隐私法（CCPA）- 美国
  • 个人信息保护法（PIPL）- 中国
  • 通用数据保护法（LGPD）- 巴西
  • 应用分发地区的其他适用隐私法律
