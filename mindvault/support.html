About NoteLM

NoteLM is a local-first AI notes app for writing notes, importing documents, recording voice notes, and asking questions against your own local library.

Frequently Asked Questions

Where is my NoteLM data stored?

NoteLM is designed as a local-first app. Your notes, chats, imported text, audio files, transcripts, and local settings are stored on your device by default. See our Privacy Policy for details.

Does NoteLM need the internet?

Most core data stays local, but some optional features use the network. Link import fetches the page you choose to import, and model downloads fetch local AI or speech models from third-party hosts.

Why is voice transcription not working?

Voice transcription needs microphone permission for recording and also requires a compatible local speech model to be available. If the model has not finished downloading yet, wait for the download to complete and try again.

What file types can I import?

Current supported import formats include TXT, MD, Markdown, JSON, CSV, XML, HTML, HTM, PDF, DOC, DOCX, ENEX, and ZIP. Directories may also be scanned recursively for supported files.

Why does a link import fail?

Link import can fail if the URL is invalid, the website blocks requests, the page has no readable text, or your network connection is unavailable. Try opening the same URL in your browser first to confirm it is reachable.

How do I enable App Lock?

Open Settings → Privacy & Security → App Lock, then create a local PIN. Keep the PIN safe. In the current implementation, the app cannot recover a forgotten PIN for you.

How do local backups work?

NoteLM can create local backup copies of its database in a backup folder on your device. Manage those backup files carefully, because they may contain sensitive note and chat data.

How do paid plans or restore purchases work?

Availability of paid features depends on the build or release channel you are using. If your version includes store-managed billing, use the app's restore option on the same store account. Some development or preview builds may show plan screens without live billing.

Troubleshooting

Import Finds No Supported Files

  1. Confirm the selected file extension is supported.
  2. If you selected a folder, confirm it actually contains importable files.
  3. Try importing one file directly instead of a whole directory.

AI Response Is Slow or Unavailable

  1. Check whether a local model is downloaded and selected.
  2. Try a smaller model if your device has limited memory.
  3. Restart the app after a model download or model switch.

Voice Note Recording Fails

  1. Allow microphone access in your system settings.
  2. Make sure the device still has enough local storage.
  3. Close and reopen the app, then try recording again.

App Lock Issues

  1. Confirm you are entering the correct local PIN.
  2. If the vault relocks after backgrounding, that is expected when App Lock is enabled.
  3. If you forgot the PIN, contact support before deleting local data so we can confirm the safest recovery path available for your build.
