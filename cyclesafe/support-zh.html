入门指南

首次设置

  1. 从 App Store 下载并安装 PetalCare
  2. 打开应用并授予必要的权限（如需要通知）
  3. 添加您的第一个月经日期开始追踪
  4. 探索高级安全功能（可选）

系统要求

  • iOS 14.0 或更高版本
  • 兼容 iPhone 和 iPad
  • 约 50MB 存储空间
  • 无需互联网连接（完全离线）

常见问题

问：如何追踪我的月经？

点击日历图标并选择月经开始日期。标记周期的每一天，应用将自动计算您的周期长度并预测未来的月经。

问：预测有多准确？

预测基于您的历史数据，随着您追踪更多周期会变得更准确。但是，许多因素可能影响您的周期，因此预测应仅用作估计。

问：我的数据安全和私密吗？

是的！您的所有数据都存储在设备本地。我们不收集、存储或传输您的任何个人或健康信息。为了额外的安全性，请考虑使用我们的高级锁定功能。

问：如何启用面容 ID/触控 ID 锁定？

这是一项高级功能。购买后，前往设置 → 安全 → 启用生物识别锁定。您需要在设备上启用面容 ID 或触控 ID。

问：我忘记了密码。如何恢复？

出于安全原因，我们无法恢复您的密码。如果您忘记了密码，需要重置应用，这将删除所有本地数据。我们建议使用面容 ID/触控 ID 作为备用身份验证方法。

问：如何导出我的数据？

前往设置 → 导出数据 → 生成 PDF。然后您可以使用 iOS 的原生分享选项分享或保存 PDF 文件。

问：我可以在多个设备之间同步数据吗？

目前，PetalCare 不支持云同步以保持最大隐私。您的数据存储在每个设备本地。您可以使用导出功能在设备之间手动传输数据。

问：如何在新设备上恢复我的购买？

在新设备上安装 PetalCare，前往设置 → 恢复购买。确保您使用的是用于原始购买的同一 Apple ID。

问：什么是伪装模式？

伪装模式（高级功能）将应用界面转换为计算器，保护您的隐私免受旁观者窥探。在设置 → 安全 → 伪装模式中启用。

问：快速锁定如何工作？

快速锁定（高级功能）允许您通过翻转手机面朝下来立即锁定应用。在设置 → 安全 → 快速锁定中启用。

故障排除

应用无法打开或崩溃

  1. 强制关闭应用并重新打开
  2. 重启您的设备
  3. 在设置 → 通用 → 软件更新中检查 iOS 更新
  4. 重新安装应用（注意：除非通过 iOS 备份，否则这将删除本地数据）

面容 ID/触控 ID 不工作

  1. 确保在 iOS 设置 → 面容 ID 与密码中启用了面容 ID/触控 ID
  2. 检查 PetalCare 是否有权使用生物识别身份验证
  3. 尝试在应用设置中禁用并重新启用该功能
  4. 重启您的设备

通知未出现

  1. 前往 iOS 设置 → 通知 → PetalCare
  2. 确保"允许通知"已启用
  3. 检查应用内的通知设置
  4. 重启您的设备

数据未保存

  1. 检查设备上的可用存储空间
  2. 确保应用具有必要的权限
  3. 尝试强制关闭并重新打开应用
  4. 如果问题仍然存在，请联系支持

购买问题

  1. 在 App Store 设置中验证您的付款方式
  2. 尝试恢复您的购买：设置 → 恢复购买
  3. 退出并重新登录您的 Apple ID
  4. 联系 Apple 支持以解决付款相关问题

数据备份与恢复

备份您的数据

PetalCare 数据会自动包含在您的 iOS 设备备份（iCloud 或 iTunes）中。要确保您的数据已备份：

  1. 对于 iCloud：设置 → [您的姓名] → iCloud → iCloud 备份 → 启用
  2. 对于 iTunes：连接到计算机并通过 iTunes/访达创建备份

恢复您的数据

当您从 iCloud 或 iTunes 备份恢复设备时，您的 PetalCare 数据将自动恢复。

隐私与安全

数据隐私

  • 所有数据都存储在您的设备本地
  • 没有数据传输到我们的服务器或第三方
  • 我们不收集任何个人或健康信息
  • 详情请参阅我们的隐私政策

安全功能

  • 生物识别锁定：使用面容 ID 或触控 ID 保护应用访问
  • 密码锁定：为应用访问设置自定义密码
  • 伪装模式：使用计算器界面隐藏应用的真实用途
  • 快速锁定：通过翻转手机立即锁定应用

高级功能

PetalCare 提供一次性购买以解锁所有高级安全功能：

  • 生物识别锁定（面容 ID/触控 ID）
  • 密码保护
  • 伪装模式（计算器）
  • 快速锁定（翻转锁定）

购买信息

  • 一次性付款（无订阅）
  • 在使用同一 Apple ID 的所有设备上可用
  • 终身访问所有高级功能
  • 未来的安全功能无需额外费用
