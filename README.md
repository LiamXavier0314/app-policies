## 应用信息

### Lansea AI - AI 陪伴
- **年龄分级**: 18+
- **类型**: AI 陪伴应用
- **功能**: 
  - 个性化 AI 陪伴对话
  - 文本和语音交互
  - 情感支持和陪伴
  - 记忆和个性化学习
  - 跨设备同步
- **内购**: 订阅制解锁高级功能（无限对话、高级 AI 能力、优先支持）
- **隐私**: 使用 RevenueCat 进行购买验证，使用第三方 AI 服务提供对话功能

### Manna - 圣经小组件
- **年龄分级**: 4+
- **类型**: 离线圣经应用
- **功能**: 
  - 中文和合本圣经内容
  - 可自定义主题和字体
  - 自定义背景图片
  - 生成和分享圣经经文图片
- **内购**: 一次性购买解锁主题、字体和背景

### CycleSafe - 月经追踪
- **年龄分级**: 12+
- **类型**: 离线月经追踪应用
- **功能**:
  - 月经周期追踪和预测
  - 症状和情绪记录
  - 数据导出（PDF）
  - 高级安全功能
- **内购**: 一次性购买解锁生物识别锁定、密码保护、伪装模式、快速锁定

## 隐私承诺

### Lansea AI
- ✅ 明确的数据收集和使用说明
- ✅ 使用 RevenueCat 进行购买验证（不处理支付）
- ✅ 使用第三方 AI 服务提供对话功能
- ✅ 对话内容不用于营销或广告
- ✅ 符合 GDPR、CCPA、PIPL、LGPD 等国际隐私法规
- ✅ 提供完整的用户数据控制权

### Manna 和 CycleSafe
- ✅ 完全离线运行
- ✅ 不收集任何用户数据
- ✅ 不使用第三方分析工具
- ✅ 所有数据仅存储在用户设备本地
- ✅ 符合 GDPR、CCPA、PIPL、LGPD 等国际隐私法规

## 部署到 GitHub Pages

1. 创建一个新的 GitHub 仓库
2. 将这些文件推送到仓库
3. 在仓库设置中启用 GitHub Pages
4. 选择主分支作为源
5. 您的网站将在 `https://yourusername.github.io/repository-name/` 上线
